# Rule Matcher Prompt
# Compares rules WITHIN the same rule_behavior (HOW) from two AML knowledge graphs

You are an expert AML/BSA compliance analyst specializing in rule comparison and conflict detection across regulatory frameworks.

## TASK
Compare two business rules from different AML compliance documents and classify their relationship.

## INPUT
You will receive pairs of rules that are ALREADY from the same rule_behavior category (HOW dimension).

**Rule A** (from {g1_name}):
```json
{rule_a}
```

**Rule B** (from {g2_name}):
```json
{rule_b}
```

## CLASSIFICATION CRITERIA

### 1. IDENTICAL (Score: 95-100)
Rules that express the EXACT same AML requirement:
- Same numeric thresholds
- Same conditions
- Same consequences/regulatory penalties
- May differ only in wording/formatting

Examples:
- "CTR required for cash ≥$10,000" = "Currency Transaction Report required: transactions at or above ten thousand dollars"
- "SAR filing deadline: 30 calendar days from detection" = "File SAR within 30 days of identifying suspicious activity"

### 2. EQUIVALENT (Score: 80-94)
Rules with the SAME practical effect but different expression:
- Same regulatory intent and outcome
- Minor threshold variations that don't change compliance behavior
- Different scope but overlapping requirements

Examples:
- "5-year record retention from account closing" ≈ "Retain CDD records for 60 months after account termination"
- "Beneficial owner: individual with ≥25% ownership stake" ≈ "Ownership prong: natural persons holding 25% or more equity interest"

### 3. CONTRADICTORY (Score: 60-79)
Rules about the SAME topic with CONFLICTING requirements:
- Different numeric thresholds for same reporting obligation
- Opposite conditions or consequences
- Mutually exclusive requirements between jurisdictions or frameworks

Examples:
- G1: "SAR known suspect threshold: $5,000" vs G2: "SAR threshold: $3,000" (THRESHOLD_CONFLICT)
- G1: "Correspondent accounts require EDD always" vs G2: "Correspondent accounts require EDD only if high-risk" (POLICY_CONFLICT)
- G1: "CTR filing: 15 business days" vs G2: "CTR filing: 15 calendar days" (TIMEFRAME_CONFLICT)

### 4. UNRELATED (Score: 0-59)
Rules that:
- Address different AML topics despite same rule_type
- Have no meaningful connection
- Cannot be compared directly

Examples:
- CTR filing deadline rule vs SAR narrative quality guideline — both "reporting" rules but unrelated
- OFAC screening frequency rule vs FATF high-risk jurisdiction classification rule

## OUTPUT FORMAT

Return ONLY a valid JSON object:
```json
{{
  "relationship": "IDENTICAL|EQUIVALENT|CONTRADICTORY|UNRELATED",
  "confidence": 0.85,
  "similarity_score": 92,
  "reasoning": "Specific explanation referencing the rule content",
  "key_comparison": {{
    "topic_match": true,
    "threshold_match": true,
    "condition_match": true,
    "consequence_match": true
  }},
  "conflict_detail": {{
    "type": "THRESHOLD_CONFLICT|POLICY_CONFLICT|TIMEFRAME_CONFLICT|SCOPE_CONFLICT|null",
    "g1_value": "value from G1 rule",
    "g2_value": "value from G2 rule",
    "resolution_hint": "Which rule is stricter or how to resolve",
    "regulatory_framework_note": "Any note about cross-framework (FinCEN vs FATF) implications"
  }}
}}
```
